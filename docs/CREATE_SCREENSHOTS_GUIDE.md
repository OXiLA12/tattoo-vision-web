# 📸 Guide Complet - Créer les Screenshots App Store

## 🎯 Objectif

Créer des screenshots professionnels pour l'App Store **depuis Windows**, sans simulateur iOS.

---

## 📐 Tailles Requises

Apple exige des screenshots pour 2 tailles d'iPhone :

| Appareil | Taille | Obligatoire |
|----------|--------|-------------|
| iPhone 6.7" (Pro Max) | **1290 x 2796** pixels | ✅ OUI |
| iPhone 6.5" (11 Pro Max) | **1242 x 2688** pixels | ✅ OUI |

**Minimum :** 3 screenshots par taille  
**Recommandé :** 5-10 screenshots par taille

---

## 🚀 MÉTHODE 1 : Captures d'Écran + Mockup (RECOMMANDÉ)

### Étape 1 : Capturer votre App Web

Votre app tourne déjà sur `http://localhost:5173`

**A. Redimensionner le Navigateur**

1. Ouvrir Chrome/Edge
2. Aller sur `http://localhost:5173`
3. Ouvrir les DevTools (F12)
4. Cliquer sur l'icône "Toggle device toolbar" (Ctrl+Shift+M)
5. Sélectionner "Responsive"
6. Entrer les dimensions : **375 x 812** (iPhone X/11/12/13)

**B. Prendre les Captures**

Naviguer vers chaque écran important et prendre des screenshots :

1. **Écran d'accueil / Bibliothèque**
   - Montrer la collection de tatouages
   - Interface principale

2. **AI Studio - Génération**
   - Champ de description
   - Bouton "Générer"
   - Exemple de prompt

3. **Résultat de Génération**
   - Tatouage généré par l'IA
   - Boutons d'action (Save, Render, etc.)

4. **Realistic Render**
   - Photo avec tatouage appliqué
   - Résultat réaliste

5. **Bibliothèque Personnelle**
   - Tatouages sauvegardés
   - Organisation

6. **Écran de Sélection de Plan** (Paywall)
   - Plans Plus, Pro, Studio
   - Prix et fonctionnalités

**Raccourci pour screenshot :**
- Windows : `Win + Shift + S` (Outil Capture d'écran)
- Ou : Extension Chrome "GoFullPage"

### Étape 2 : Créer les Mockups iPhone

**Option A : Utiliser Figma (GRATUIT - Recommandé)**

1. **Créer un compte Figma**
   - https://www.figma.com/
   - Gratuit

2. **Utiliser un Template iPhone**
   - Chercher "iPhone mockup" dans Figma Community
   - Ou utiliser ce template : https://www.figma.com/community/file/1234567890/iphone-mockup

3. **Créer vos Frames**
   ```
   Frame 1 : 1290 x 2796 (iPhone 6.7")
   Frame 2 : 1242 x 2688 (iPhone 6.5")
   ```

4. **Importer vos Captures**
   - Glisser-déposer vos screenshots
   - Redimensionner pour remplir l'écran iPhone

5. **Ajouter du Texte Marketing (Optionnel)**
   ```
   Screenshot 1 : "Créez des tatouages uniques avec l'IA"
   Screenshot 2 : "Visualisez sur votre corps en temps réel"
   Screenshot 3 : "Bibliothèque complète incluse"
   Screenshot 4 : "Rendus ultra-réalistes"
   Screenshot 5 : "Choisissez votre plan"
   ```

6. **Exporter**
   - Sélectionner chaque frame
   - Export → PNG
   - 2x ou 3x pour haute qualité

**Option B : Utiliser Canva (GRATUIT)**

1. **Créer un compte Canva**
   - https://www.canva.com/
   - Gratuit

2. **Créer un Design Personnalisé**
   - Dimensions : 1290 x 2796
   - Puis : 1242 x 2688

3. **Ajouter un Template iPhone**
   - Chercher "iPhone mockup" dans les templates
   - Ou créer un rectangle avec les bonnes dimensions

4. **Importer vos Screenshots**
   - Uploads → Vos captures d'écran
   - Placer dans le mockup

5. **Ajouter du Texte**
   - Titre accrocheur
   - Description courte
   - Utiliser les couleurs de votre app (bleu cyan)

6. **Télécharger**
   - Format PNG
   - Haute qualité

**Option C : Utiliser un Outil en Ligne**

## 🚀 MÉTHODE 3 : Screenshots Simples (Rapide)

Si vous voulez aller vite, vous pouvez soumettre des screenshots simples sans mockup iPhone :

1. **Redimensionner vos Captures**
   - Utiliser un outil comme https://www.iloveimg.com/resize-image
   - Redimensionner à 1290 x 2796
   - Ajouter des bordures si nécessaire

2. **Ajouter un Fond**
   - Fond uni (noir ou dégradé bleu)
   - Centrer votre capture

⚠️ **Moins professionnel, mais acceptable pour Apple**

## 🎨 Template Figma Rapide

Si vous utilisez Figma, voici un template simple :

```
1. Créer un Frame : 1290 x 2796
2. Fond : Dégradé noir → bleu foncé
3. Rectangle pour l'écran : 375 x 812 (centré)
4. Importer votre screenshot dans le rectangle
5. Texte au-dessus :
   - Titre : 48px, Bold, Blanc
   - Sous-titre : 24px, Regular, Gris clair
6. Dupliquer pour chaque screenshot
7. Exporter tout en PNG 2x
```

## 📊 Ordre des Screenshots Recommandé

Apple affiche les screenshots dans l'ordre où vous les uploadez. Voici l'ordre optimal :

```
1. Realistic Render (WOW factor - montrer le résultat final)
```

## 🆘 Problèmes Courants

### "Mes screenshots sont flous"
```
Solution :
- Exporter en 2x ou 3x depuis Figma
- Utiliser PNG (pas JPG)
- Vérifier que la résolution est exacte
```

### "Le texte est trop petit"
```
Solution :
- Utiliser minimum 24px pour le corps
- Minimum 48px pour les titres
- Tester la lisibilité sur mobile
```

### "Apple rejette mes screenshots"
```
Raisons communes :
- Mauvaise taille (vérifier les pixels exacts)
- Contenu offensant
- Texte illisible
- Screenshots ne correspondent pas à l'app

Solution :
- Vérifier les dimensions exactes
- Utiliser du contenu approprié
- Agrandir le texte
```
