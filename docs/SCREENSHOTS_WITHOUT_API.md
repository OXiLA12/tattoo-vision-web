# 📸 Guide Screenshots - Sans API Fonctionnelles

## ✅ **C'est Normal !**

Les fonctionnalités IA (Gemini, remove.bg) ne fonctionnent pas en local parce que :
1. Les Edge Functions Supabase ne tournent pas localement
2. Les clés API (Gemini, remove.bg) sont dans les secrets Supabase
3. C'est normal et attendu !

---

## 🎯 **Solution : Screenshots avec Mockups**

Vous allez créer des screenshots "simulés" mais **réalistes** pour l'App Store.

### **Écrans Qui Fonctionnent (Sans API)**

✅ **Ces écrans marchent déjà :**

1. **Écran d'accueil**
   - Navigation
   - Menu

2. **Bibliothèque de tatouages**
   - Galerie de tatouages gratuits
   - Catégories

3. **Profil / Paramètres**
   - Informations utilisateur
   - Paramètres

4. **Écran de connexion**
   - Formulaire de login

5. **Paywall / Plans**
   - Sélection de plan
   - Prix

---

## 📸 **Plan de Screenshots Recommandé**

### **Screenshots à Prendre DIRECTEMENT (Fonctionnent)**

#### 1. **Bibliothèque de Tatouages**
```
1. Aller sur http://localhost:5173
2. Naviguer vers la bibliothèque
3. F12 → Ctrl+Shift+M → 375 x 812
4. Win + Shift + S
```

**Pourquoi :** Montre la variété de tatouages disponibles

#### 2. **Écran de Sélection de Plan (Paywall)**
```
1. Naviguer vers les plans d'abonnement
2. Screenshot avec les 3 plans (Plus, Pro, Studio)
```

**Pourquoi :** Montre les options d'abonnement

#### 3. **Écran d'Accueil**
```
1. Page d'accueil de l'app
2. Navigation principale
```

**Pourquoi :** Première impression

---

### **Screenshots à SIMULER (Nécessitent API)**

Pour les fonctionnalités IA, vous allez créer des mockups :

#### 4. **AI Studio - Interface de Génération**

**Méthode :**
```
1. Prendre screenshot de l'interface AI Studio
2. Ajouter un exemple de prompt dans le champ texte :
   "Dragon japonais avec fleurs de cerisier, style traditionnel"
3. Screenshot AVANT de cliquer sur Générer
```

**Résultat :** Montre l'interface de génération

#### 5. **Résultat de Génération IA**

**Méthode avec Figma/Photoshop :**
```
1. Prendre screenshot de l'interface de résultat (vide)
2. Trouver une image de tatouage de dragon (Google Images, Pinterest)
3. Avec Figma :
   - Importer le screenshot
   - Ajouter l'image du tatouage à la place du résultat
   - Exporter
```

**Ou utiliser cette méthode simple :**
```
1. Trouver une app similaire sur l'App Store
2. S'inspirer de leurs screenshots
3. Recréer avec votre interface
```

#### 6. **Realistic Render**

**Méthode :**
```
1. Trouver une photo de bras/jambe (Unsplash, Pexels - gratuit)
2. Trouver un tatouage transparent (PNG)
3. Avec Photoshop/Figma/Canva :
   - Superposer le tatouage sur la photo
   - Ajuster l'opacité pour réalisme
4. Importer dans votre interface
5. Screenshot
```

**Ressources gratuites :**
- Photos : https://unsplash.com/ (chercher "arm" ou "leg")
- Tatouages PNG : https://www.pngwing.com/ (chercher "tattoo")

## 🎨 **Méthode Alternative : Tout Créer dans Figma**

Si vous voulez aller plus vite, créez TOUS les screenshots dans Figma :

### **Template Figma Rapide**

1. **Créer un compte Figma** (gratuit)
   - https://www.figma.com/

2. **Créer un Frame iPhone**
   ```
   - Taille : 1290 x 2796 (iPhone 6.7")
   - Fond : Dégradé noir → bleu foncé
   ```

3. **Pour chaque screenshot :**

**Screenshot 1 - Bibliothèque**
```
- Titre : "Des milliers de tatouages"
- Grille d'images de tatouages (Google Images)
- Interface de navigation
```

**Screenshot 2 - AI Studio**
```
- Titre : "Créez avec l'IA"
- Champ texte avec exemple de prompt
- Bouton "Générer"
```

**Screenshot 3 - Résultat IA**
```
- Titre : "Designs uniques"
- Image d'un tatouage de dragon
- Boutons : Save, Render, Share
```

**Screenshot 4 - Realistic Render**
```
- Titre : "Visualisez sur vous"
- Photo de bras avec tatouage appliqué
- Slider d'ajustement
```

**Screenshot 5 - Ma Bibliothèque**
```
- Titre : "Votre collection"
- Grille de tatouages sauvegardés
```

**Screenshot 6 - Plans**
```
- Titre : "Choisissez votre plan"
- 3 cartes : Plus, Pro, Studio
- Prix et fonctionnalités
```

4. **Exporter tout**
   - Sélectionner tous les frames
   - Export → PNG → 2x

## 🎯 **Workflow Recommandé (1 heure)**

### **Option A : Mix Réel + Mockup (Recommandé)**

**Temps total : 1 heure**

```
1. Screenshots réels (20 min)
   - Bibliothèque de tatouages
   - Plans d'abonnement
   - Écran d'accueil

2. Trouver des ressources (15 min)
   - 1 photo de bras (Unsplash)
   - 3-4 images de tatouages (Pinterest/Google)

3. Créer mockups dans Figma (20 min)
   - AI Studio interface
   - Résultat de génération
   - Realistic render

4. Finaliser et exporter (5 min)
   - Redimensionner si nécessaire
   - Organiser dans screenshots/
```

### **Option B : Tout dans Figma (Plus rapide)**

**Temps total : 45 min**

```
1. Créer 6 frames dans Figma (10 min)
2. Importer images et texte (20 min)
3. Ajouter texte marketing (10 min)
4. Exporter (5 min)
```

## 💡 **Conseil Pro**

**Apple ne vérifie PAS si les screenshots sont "réels"**

Tant que :
- ✅ Les screenshots représentent fidèlement les fonctionnalités de l'app
- ✅ Pas de contenu trompeur
- ✅ Bonne qualité visuelle

**C'est accepté !**

Beaucoup d'apps utilisent des mockups et des simulations pour leurs screenshots App Store.
