## 🚀 Comment Publier Ces Documents

### Option 1 : GitHub Pages (Gratuit)

1. **Créer un repo GitHub** : `tattoovision-legal`

2. **Créer les fichiers** :
   - `privacy.md`
   - `terms.md`

3. **Activer GitHub Pages** :
   - Settings → Pages
   - Source : main branch
   - Thème : Minimal

4. **URLs finales** :
   - `https://[votre-username].github.io/tattoovision-legal/privacy`
   - `https://[votre-username].github.io/tattoovision-legal/terms`

### Option 2 : Site Web Simple

Créer un site avec :
- `tattoovision.com/privacy`
- `tattoovision.com/terms`

### Option 3 : Notion Public

1. Créer deux pages Notion
2. Les rendre publiques
3. Copier les liens publics
